# Rofi WiFi

> [!NOTE]
> This is a personal project created for fun to mess with the [Odin](https://odin-lang.org/) programming language.

A simple WiFi menu using [Rofi](https://github.com/davatorium/rofi).

## Prerequisites

- rofi
- nmcli
- odin

## Getting Started

Run the program:

```sh
odin run .
```
